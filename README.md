# Sudoku em Python

Este repositório contém uma implementação em Python de um Sudoku. Ele pode receber qualquer quebra-cabeça válido de Sudoku como entrada para que o usuário possa resolvê-lo.

## Funcionalidades

- Suporta qualquer grade de Sudoku válida (grade 9x9).
- Pode ser resolvido pelo usuário.
- Exibe a grade de Sudoku de forma amigável ao usuário.

## Requisitos

Para rodar o solucionador de Sudoku, você precisa ter o Python 3 instalado no seu sistema. Além disso, a implementação utiliza a biblioteca pygame.

## Instalação

1. Clone este repositório para sua máquina local:
2. Acesse o código fonte em "cd main"
3. dê um pip install pygame
4. digite python Sudoku.py para jogar.
